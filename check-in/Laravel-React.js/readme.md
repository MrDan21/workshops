# Check-in
Assistance control web application through qr code with Laravel + React.js

# Installation

`git clone https://github.com/MrDan21/workshops.git`

`cd Laravel-React.js folder`

`cp .env.example .env`

`composer install`

`npm install`

`php artisan key:generate`

`php artisan migrate`

# Usage

### Development

`npm run watch`

### Production

`npm run production`

# Requeriments

PHP >= 7.2
